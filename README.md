# Clash Royale Trainer

## Authors
- Benjamin Dohna

## Description
A simple training tool for improving strategic decision-making in Clash Royale.  
The project begins with a terminal-based elixir quiz and will grow into a web-based platform with interactive exercises.

## Purpose
Help players improve:
- Elixir awareness  
- Card recognition  
- Evaluating elixir trades  
- Placement accuracy (future)

## Current Features
- JSON card database  
- Terminal quiz for elixir costs
